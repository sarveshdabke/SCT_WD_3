## Project Description
This project is a fully functional and interactive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript.  
The game allows users to play against another player or against the computer while tracking game state and checking winning conditions dynamically.

## Features
- Interactive 3x3 Tic-Tac-Toe board
- Player vs Player mode
- Player vs Computer mode
- Real-time game state tracking
- Automatic win and draw detection
- Prevents invalid moves
- Reset / restart game functionality
- Responsive design for all screen sizes
- Smooth hover and click animations

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation & Event Handling)

## Key Learnings
- Handling user click events using JavaScript
- Managing and updating game state dynamically
- Implementing winning and draw condition logic
- DOM manipulation for real-time UI updates
- Writing clean and readable JavaScript code

## How to Run the Project
1. Download or clone the repository.
2. Open the `TicTacToe.html` file in any modern web browser.
3. Select the game mode and start playing.
